## Problema 1/5: Group Anagrams

**Dificuldade:** Medium | **Padrao:** HashMap + Sorting | **Tempo alvo:** 15 min

### Enunciado

Dada uma lista de strings, agrupe as strings que sao anagramas entre si.

Duas strings sao anagramas se contem as mesmas letras na mesma frequencia (ex: "eat" e "tea").

### Input/Output

```
Input:  ["eat", "tea", "tan", "ate", "nat", "bat"]
Output: [["eat", "tea", "ate"], ["tan", "nat"], ["bat"]]
```

A ordem dos grupos e dentro dos grupos nao importa.

### Constraints

- `1 <= len(strs) <= 10^4`
- `0 <= len(strs[i]) <= 100`
- `strs[i]` contem apenas letras minusculas (a-z)

### Exemplos adicionais

```
Input:  [""]
Output: [[""]]

Input:  ["a"]
Output: [["a"]]
```

**Timer: 15 min.** Resolve e cola a solucao. Se travar por mais de 5 min, pede dica.
